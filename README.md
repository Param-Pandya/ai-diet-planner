<h2>🚀 Installation</h2>

Clone the repository

```bash
git clone https://github.com/Param-Pandya/ai-diet-planner.git
```

Navigate to the project

```bash
cd ai-diet-planner
```

Install dependencies

```bash
npm install
```

Run Expo

```bash
npx expo start
```

Open **Expo Go** on your mobile device and scan the QR code.

---

<h2>🔑 Gemini API Configuration</h2>

1. Generate an API Key from **Google AI Studio**

2. Open the application

3. Navigate to

```text
Settings
      ↓
API Key
```

4. Paste your Gemini API Key

5. Save
